7starmovies Demo (20 movie posts)
---------------------------------
How to use:
1. Unzip and upload the folder contents to your GitHub repository root.
2. Replace images in the images/ folder with your real posters keeping the filenames the same.
3. Edit movies/*.html files to replace the demo '#' link with real download/watch links.
4. Commit & push to GitHub. Netlify will auto-deploy (or use any static host).
5. Open index.html to see the movie cards. Use the search box to filter titles.

Folder structure:
- index.html
- /movies/*.html
- /images/*.jpg
- /css/style.css
- /js/script.js
